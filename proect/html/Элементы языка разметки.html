Элемент HTML

Это основная структурная единица веб-страницы, написанная на языке HTML. Данный элемент вы можете увидеть в исходном коде для всех веб-страниц после задания типа документа на первой строке на странице. DOCTYPE определяет, какую версию (X)HTML эта страница использует. Элементы страницы находятся между открывающим тегом HTML и закрывающим HTML. Элемент html называется корневым элементом.
