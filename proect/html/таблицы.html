table (от англ. table — таблица) служит контейнером для элементов, определяющих содержимое таблицы. Любая таблица состоит из строк и ячеек, которые задаются с помощью элементов tr и td. Внутри table допустимо использовать следующие элементы: caption, col, colgroup, tbody, td, tfoot, t>, thead и tr.
