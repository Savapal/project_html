Введение в HTML

HTML (Hypertext Markup Language)

это код, который используется для структурирования и отображения веб-страницы и её контента. Например, контент может быть структурирован внутри множества параграфов, маркированных списков или с использованием изображений и таблиц данных.

HTML документ

HTML-документ создаётся в обычном текстовом редакторе — зачастую в "Блокноте". Так же есть и специализированные приложения: Visual Studio Code, Sublime Text, WebStorm и многие другие — с подсветкой кода.

Минимальная страница

HTML документ состоит из тегов, которые указывают как отбражать элементы на странице. Минимальная страница состоит из тегов: 

            
              <html>
                  <head>
                  </head>
                  <body>
                  </body>
              </html>

Теги html

тег html являются контейнерами для содержимого страницы/приложения

Блочные элементы

блочные элементы - такие элементы отображаются блоками к блочным элементам относятся теги: div пустой блок. p параграф. ol нумерованный список. ul маркированный список. h1-h6 заголовки разного размера. address предназначен для хранения контактной информации автора. blockquote для цитат. feildset предназначен для группирования элементов формы. Такая группировка облегчает работу с формами. form создает поля, которые можно заполнять. hr рисует горизонтальную линию. table создает таблицу.

        универсальные элементы - может быть использован как блочный так встраиваемый элемент

        object
        Сообщает браузеру, как загружать и отображать объекты, которые исходно браузер не понимает.
        
        iframe
        Создает плавающий фрейм.
        
        button
        Создает на веб-странице кнопку.
        
        applet
        Предназначен для вставки на страницу апплетов — небольших программ, написанных на языке Java.
        
        ins
        Предназначен для выделения текста, который был добавлен в новую версию документа.
        
        del
        Используется для выделения текста, который был удален в новой версии документа.
